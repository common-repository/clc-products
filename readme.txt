A plugin that adds product info and links to products that are found on clcnederland.com.

== Description ==

This plugin adds products found on clcnederland.com to your wordpress website with an affiliate link to clcnederland.com. This makes this plugin ideal if you want to add a book, cd or dvd to your article and are participating in our affiliate program. But even if you don't participate in our affiliate program you can use this plugin to add a product to your article. 

The plugin makes use of an API on boekenenmuziek.nl (part of CLC Nederland). The API is, free and without dependencies, accesible on https://api.boekenenmuziek.nl. 

For our terms of use and privacy policy: Visit https://api.boekenenmuziek.nl/ 
If you want to use this plugin as an affiliate partner, we ask you to contact us on email (webshop@clcnederland.com)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/clc-product` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->CLC Product screen to configure the plugin
4. Make use of the shortcode on the Settings->CLC Product screen to show a product from clcnederland.com on your site or add a product using the Gutenberg blocks
